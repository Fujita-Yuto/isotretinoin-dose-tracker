# イソトレチノイン累積投与量計算

**https://isotretinoin-dose-tracker.vercel.app/**

イソトレチノイン（アキュテイン/ロアキュタン等）を服用中の方向けに、累積投与量（mg/kg）を自動計算・記録できる無料のWebアプリです。

## 特徴

- 💊 **累積投与量の自動計算** — 服用記録（開始日と1日量）と体重を入れるだけで、累積量 (mg/kg)・目標までの進捗・到達予測日を表示
- 📈 **推移グラフ** — 治療開始からの累積量の推移を折れ線グラフで確認
- 📅 **服用カレンダー** — 飲み忘れをタップで記録すると計算から自動除外
- ⚖️ **2つの計算モード** — 最新体重によるシンプル計算 / 期間ごとの体重を使った厳密計算
- 🔒 **完全クライアントサイド** — データは端末のブラウザ内（localStorage）にのみ保存され、サーバーには一切送信されません。登録不要・広告なし
- 📲 **PWA対応** — ホーム画面に追加するとアプリとして起動、オフラインでも動作
- 💾 **バックアップ** — JSONファイルでのエクスポート/インポート（機種変更対応）

## 免責事項

本アプリは計算補助ツールであり、医療アドバイスではありません。記載内容は医師の監修を受けたものではなく、計算結果は参考情報です。服用量の変更・中止・再開などの判断は、必ず医師の指導のもとで行ってください。

## 技術構成

- Vite + React + TypeScript
- Tailwind CSS v4
- 計算ロジックは純関数として分離し、Vitest でユニットテスト（46件）
- Vercel でホスティング、vite-plugin-pwa で PWA 化

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm test         # ユニットテスト
npm run build    # 本番ビルド（dist/）
```
